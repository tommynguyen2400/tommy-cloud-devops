# Tommy Portfolio HTML

Website cá nhân dạng portfolio, lấy cảm hứng từ layout section: Giới thiệu, Hành trình/Kinh nghiệm, Kỹ năng, Bộ sưu tập, Liên hệ.

## Chạy local

```bash
python3 -m http.server 8080
```

Mở: http://localhost:8080

## Deploy GitHub Pages

Push toàn bộ thư mục này lên repo, sau đó vào Settings > Pages > Deploy from a branch > main > /root.
